HWPX Filler — UI 프로토타입

UI 설계 핸드오프와 표면 결정 기록(ADR A~I)에서 도출한 투트랙 UI — HWPX(대시보드· 에디터·실행)와 txt(즉시 기안). 색·위젯은 gui/style.py 팔레트를 따르고, 각 요소의 data-vm은 실제 ViewModel 계약에 배선됩니다(UI_CONTRACT.md).

왜 /design-sync가 아니라 프로토타입인가. design-sync는 컴파일된 JS/React 디자인 시스템(dist/+Storybook/npm)을 claude.ai/design에 올려 디자인 에이전트가 실제 컴포넌트로 빌드하게 하는 스킬입니다. 이 저장소는 Python·PySide6 데스크톱 앱이라 올릴 컴포넌트 라이브러리가 없습니다 — 그래서 confirm-or-alarm 원칙대로 억지 실행 대신, 설계 문서를 읽어 화면을 시각 디자인으로 옮겼습니다.
