R-FLOW 블록 4 본안 · 가드·연속성 시연 v1.3 · 합의 종결

가드는 미수 노동을 지키고, 정의는 세션을 건넌다

확정된 ①~③(스코프·전이 원장·가드 형식) 위에서, 제안 상태인 ④의 실물을 걷습니다 — 공고서 세션의 필터 정의가 인계용 값 객체로 살아남아 기안 세션의 「직전 필터 재적용」 칩이 되는 흐름. 가드 모달(수치 재진술)·완료 무장해제· 마법사 비파괴화 고지·열 결손 부분 설치가 전부 동작합니다.

1. 걸어보기: 공고서 → 기안, 같은 건의 두 출력

권장 동선 — ① 상태 버튼으로 공고서 세션을 만들고 ② 좌측 「물품 구매 기안」을 클릭해 전환 반응(무발화/모달)을 확인, ③ 기안 세션에서 보라 점선의 재적용 칩 → 「전체 선택」으로 큐 선언, ④ 「복사」로 큐를 진행하다 행을 전환해 T3 가드 확인. 「용역 정산 기안」은 데이터 소스가 달라 칩 자체가 나타나지 않습니다 (v1.2 소스 일치 게이트). 「마법사 완료」 버튼은 D2 비파괴화를 시연합니다.

공고서 세션 상태
㉠가드 술어 = 재현 불가능성(B1) — 선택이 정의-유래 집합과도, 마지막 생성·완주 집합과도 다를 때만 선택 성분이 선다. 상태 A(정의-유래)와 상태 C(생성 완료)의 전환은 조용히 통과하고, 상태 B만 모달이 선다. 더티 비트가 아니라 집합 비교라 켰다-끈 순변화 0도 오발화하지 않는다.
㉡수치 재진술 — 모달은 실재하는 성분만 문장에 올린다(수동 선택 N건 · 입력 필드 M개 · 복사 진행 M/N건). 무엇을 만들지의 게이트는 표본 열거가 필요했지만, 무엇을 잃는지의 고지는 종류와 규모면 충분하다. 기본 포커스=「머무르기」, Escape=머무르기.
㉣재적용 칩(3안) — 직전 세션의 필터 정의(직전 1개·세션 메모리)가 있고, 현 필터가 비어 있고, 직전 정의의 데이터 소스가 현 작업과 같을 때만 나타나는 명시 동사(v1.2). 소스별 맵은 "전달"이 "기억"으로 변질되는 뒷문이라 기각 — 슬롯 1개·수명 규칙 하나. 소스 교차는 재타이핑 몇 초로 족하다(차등 논거). 저장이 아니라 전달 — 작업 무기억 비침해.
㉥재적용 ↔ 선택 분리 — 칩은 정의 설치까지만 하고, 큐 선언은 「전체 선택」으로 사용자가 한다. 하나는 보기의 복원, 하나는 게이트로 이어지는 선언 — 의사표시의 격이 달라 2클릭을 유지한다 (블록 1 "포커스=첫 미답 질문"과 동형).
㉤열 결손 = 백스톱으로 강등(v1.2) — 소스 일치 게이트가 서면 열 결손은 설계 경로에서 사라지고, 세션 사이에 대장 파일의 열이 외부 편집으로 사라진 드문 edge만 남는다. 그때는 부분 설치+탈락 가지 고지, 설치 가능한 가지 0이면 거부+이유(코드 잔존). 조용한 부분 적용만 아니면 confirm-or-alarm이 지켜진다.
㉢T2 비파괴화(D2) — 마법사 완료 시 진행 중 세션이 있으면 자동 선택을 애초에 생략하고 조용한 고지 한 줄을 남긴다. 파괴를 확인으로 지키는 게 아니라 전이 자체를 비파괴로 만든다(Explorer 기각과 같은 상류 제거). 새 작업은 강조 표시로 위치만 알린다.

2. 이 시연이 묻는 것

④ 제안: 기제 서열 = 영속성 사다리의 아래부터. 3안 정의 재적용 즉채택(무영속·무객체·A3의 프루닝 복원 겸용) → 2안 선택 집합 직접 인계는 "수동 열거를 큐로 넘겨야 하는 마찰"의 실사용 실증 시 → 1안 묶음 즐겨찾기는 1부 아카이브 착수 조건(짝-날 재선택 마찰 실증) 유지, 채택 시 2안을 흡수. 위 칸의 착수 조건은 아래 칸의 갭 실증이다.

?확정 요청 3 + 시연 발견 1

  • 서열 그 자체 — 3안 즉채택·2안/1안 조건부 유보가 이 실물로 성립하는가.
  • 재적용 ↔ 전체 선택 분리(㉥) — 2클릭이 마찰인가 의사표시인가. 붙이면 재적용이 곧 큐 선언이 된다.
  • 소스 일치 게이트(㉣, v1.2) — 소스 분기 난점의 해소로 노출 조건에 소스 일치를 추가. 슬롯 1개 유지·소스별 맵 기각. 열 결손 처우(㉤)는 백스톱으로 강등.
  • 술어 정련(㉠, 시연 중 발견) — ③의 완료 무장해제를 B1과 정합시키려면 선택 성분은 "≠ 정의-유래 ∧ ≠ 마지막 생성·완주 집합"이어야 한다. 생성이 끝난 수동 열거는 미수 노동이 아니다. ③ 확정문에 이 절 추가를 제안.

…미결 (구현 라운드로 넘어가는 것)

  • 재적용 후 프루닝 상태 — 시연은 프루닝된 가지를 정의의 일부로 함께 나른다 (A3의 취지). 우회 잔여 "프루닝의 지속성"(텍스트 수명 문제)과 접합.
  • 재적용 칩의 수명 — 시연은 설치 후 숨김·정의 소거 시 재노출. 새 검색으로 그룹을 덮어쓰면 직전 정의도 그 새 정의로 교체된다(직전 1개 원칙).
