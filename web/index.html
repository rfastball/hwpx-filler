HWPX Filler

대시보드

저장된 작업(HWPX)과 기안 템플릿(txt)을 한눈에.

HWPX 문서 만들기

누름틀 템플릿과 매핑을 작업으로 저장하고, 데이터를 골라 .hwpx 문서를 만듭니다.

기안 · txt

평문 {{필드}} 템플릿을 실시간으로 채워 복사합니다. 온나라 기안 등에 바로 붙여넣을 수 있습니다.

작업

저장된 작업을 골라 데이터를 채워 문서를 만듭니다.

작업 선택
문서 작업 · HWPX
저장된 작업이 없습니다.
왼쪽에서 작업을 선택하면 세션 패널이 열립니다.
1템플릿·헤더 확인
2데이터 연결
데이터(.xlsx/.csv)
생성 대상 문서
선택 0/0
데이터를 선택하면 생성 대상 문서가 여기에 표시됩니다.
3본문 확인·거울

이 문서에 들어갈 값입니다. 빈 값 행은 눌러서 확인해야 생성할 수 있습니다.

저장 폴더
4생성
작업, 데이터, 저장 폴더를 확인한 뒤 생성하세요.

기안 TXT

저장된 기안을 고르거나, 그냥 템플릿을 붙여넣어 채워 복사합니다.

기안 선택
기안 작업 · TXT
저장된 기안 작업이 없습니다.
1데이터
템플릿
데이터(.xlsx/.csv)
기안 대상 행
선택 0/0
데이터를 연결하면 여러 건을 한 번에 채웁니다. 없이도 '맞추기'에서 직접 채울 수 있습니다.
2맞추기
3이대로 복사됩니다
